# dsa

Project workspace with Cursor agent skills.

## Skills

| Skill | Description |
|-------|-------------|
| [grill-me](.agents/skills/grill-me/SKILL.md) | Stress-test plans and designs through systematic one-at-a-time questioning |

Use `/grill-me` in Cursor, or ask the agent to grill you on a plan or design.

## Install skills

From `skills-lock.json`:

```bash
npx skills experimental_install
```

Update:

```bash
npx skills update grill-me
```
